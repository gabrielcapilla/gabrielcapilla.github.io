## Sync Commands

### push

Link all files from the profile to the system.

<div class="code-block"><pre>dotman push &lt;profile&gt;</pre></div>

### pull

Remove all symbolic links from the selected profile.

<div class="code-block"><pre>dotman pull &lt;profile&gt;</pre></div>
