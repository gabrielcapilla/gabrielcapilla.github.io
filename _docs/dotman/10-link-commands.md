## Link Commands

### add

Create a symlink from the profile to the system. The file must exist in the profile directory.

<div class="code-block"><pre>dotman add &lt;file&gt;
dotman -a &lt;file&gt;</pre></div>

### remove

Remove a symlink from the profile.

<div class="code-block"><pre>dotman remove &lt;file&gt;
dotman -r &lt;file&gt;</pre></div>
