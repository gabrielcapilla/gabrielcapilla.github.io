## Profile Commands

### init

Initialize dotman directory structure in ~/.dotfiles.

<div class="code-block"><pre>dotman init
dotman -i</pre></div>

### make

Create a new profile with the specified name.

<div class="code-block"><pre>dotman make &lt;name&gt;
dotman -m &lt;name&gt;</pre></div>

### clone

Clone an existing profile to create a new one.

<div class="code-block"><pre>dotman clone &lt;source&gt; &lt;destination&gt;
dotman -c &lt;source&gt; &lt;destination&gt;</pre></div>

### list

List all available profiles.

<div class="code-block"><pre>dotman list
dotman -l</pre></div>
