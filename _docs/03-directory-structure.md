## Directory Structure

By default, dotman uses `~/.dotfiles` as the root directory. The structure is organized by **categories** that map to specific system locations:

<div style="background: #0a0e17; padding: 1.5rem; border-radius: 0.5rem; border: 1px solid rgba(255,255,255,0.1); font-family: var(--font-mono); color: var(--text-muted); margin-bottom: 2rem;">
    <div><span class="text-blue">~/.dotfiles/</span></div>
    <div style="padding-left: 1rem;">└── <span class="text-blue">main/</span> <span class="text-gray"># Default profile</span></div>
    <div style="padding-left: 2rem;">├── <span class="text-blue">config/</span></div>
    <div style="padding-left: 2rem;">├── <span class="text-blue">share/</span></div>
    <div style="padding-left: 2rem;">├── <span class="text-blue">home/</span></div>
    <div style="padding-left: 2rem;">└── <span class="text-blue">bin/</span></div>
</div>
